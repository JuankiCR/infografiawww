Historia.

Tim Berners-Lee Creador de la World Wide Web

Empezó en América, Estados Unidos, en 1969 era una simple intercomunicación para la transferencia de textos e informaciones entre organismos de carácter militar o instituciones educativas universitarias.

El proyecto nació en la órbita del CERN, la organización europea de investigación nuclear.

1992 Primera imagen que se colgó a la web fue una fotografía de la banda "Les Horribles Cernettes".

Nathaniel Boronstein y Ned Freed desarrollaron la función que permite adjuntar archivos a los correos electrónicos.

1993 CERN decidió entregar los códigos de la Web al dominio público.

Características.

Web 1.0.

El usuario no puede interactuar con el contenido de la página estando totalmente limitado a lo que el Webmaster sube a la página web.

  • Para publicar requiere del webmaster.
  • El usuario tiene un rol pasivo.
  • Estática con documentos que jamás se actualizaban.
  • Predominio del texto sobre la imagen que suele estar en formato gif.
  • Uso del formato HTML para poder guardar y crear las páginas.
  • Navegadores de solo texto (solo lectura).

Web 2.0.

Evolución de la Web o Internet en el que los usuarios dejan de ser usuarios pasivos para convertirse en usuarios activos.

  • Web colaborativa
  • Permite a los usuarios interactuar y colaborar entre sí.
  • Permite a estudiantes y docentes mejorar las herramientas utilizadas en el aula de clase.
  • Aplicaciones web dinámicas.

Se crearon herramientas como:

  • Wikis -> Colaborativas
  • Blogs -> Edición masiva de información
  • Comunicación -> Redes sociales

Web 3.0.

Potenciación de creación de nuevas formas de ocio y comunicación entre usuarios.

  • WEB semántica.
  • Utilización de inteligencia artificial, en combinación con la nueva estructura.
  • Potenciación de creación de nuevas formas de ocio y comunicación entre usuarios.
  • Prevalencia del usuario en la creación, rendimiento y organización del contenido.

Web 4.0.

Sistema operativo a nivel mundial con mayor rapidez y eficacia

  • Avance más grande de las comunicaciones
  • Aparición de los metadatos
  • Campo de las predicciones
  • Facilita la investigación
  • Sistema operativo de web global tan rápido en respuesta como la del ser humano
  • Avance de la nanotecnología en el mundo
